Accessibility Highlights

  • All interactive controls expose aria- metadata and focus states.
  • Theme switcher maintains AAA contrast for body text and AA for chrome.
  • Reduced motion toggle respects system preferences and updates animation helpers.
  • Audio mute toggle honors user preference and prefers-reduced-motion for quiet experiences.
  • Menus, tabs, and segmented controls support keyboard roving indexes.
